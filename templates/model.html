Network Specifications

Our handwriting deep learning character classifier is implemented as a sequential Keras neural network. OpenCV splits coordinate blocks before feeding pixels box-by-box.

Batch Normalization

Optimizes parameters internally, accelerating gradient convergence rates across epochs.

Conv2D Filter maps

Transitions from 32 input kernels to 128 deep layers, extracting edges and stroke connections sequentially.

Page-Level OCR Engine

For notebook page extraction, we leverage a pre-trained **EasyOCR** engine containing ResNet and LSTM modules to scan sentences and line breaks.
