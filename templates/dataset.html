Training Databases Overview

Description of MNIST, EMNIST, and HTR datasets used to configure the network weights.

MNIST Digits

The **MNIST database** is a collection of handwritten digits. It contains 70,000 grayscale images of size 28×28 pixels representing values 0 to 9.

  • Training samples: 60,000
  • Test samples: 10,000
  • Channels: 1 (Grayscale)

EMNIST Balanced

The **EMNIST (Extended MNIST)** dataset expands digits to English letters (A–Z, a–z), mapping 47 balanced output categories total.

  • Total samples: 131,600
  • Categories: 47 classes
  • Resolution: 28×28 pixels

IAM HTR Dataset

The **IAM Handwriting Database** contains complete forms of handwritten English texts scanned from notebook pages, used to train page-level OCR engines.

  • Number of writers: 657 writers
  • Text structures: Paragraph lines
  • Classifier engine: PyTorch EasyOCR
