Handwritten Text Recognition
Using Deep Learning

A clean, professional academic web application demonstrating computer vision character segmentation, Keras neural network predictions (EMNIST/MNIST), and page-level handwriting text extraction. Developed as an MCA final-year capstone project.

Access Dashboard Console Project Specifications

System Modalities

Digits & Alphabets

Sketch or upload isolated characters. The Keras model predicts digits (0–9) and alphabets (A–Z, a–z) with instant probability charts.

Word & Line Segmenter

Write complete words. The OpenCV preprocessing segmenter detects letter boundaries, runs classifications, and joins characters.

Full Page OCR Scan

Upload scans of notebooks or text blocks. The advanced neural OCR engine extracts sentences, preserving lines and paragraphs.
