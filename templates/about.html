About the Project

This **Handwritten Text Recognition (HTR) System** represents a complete, presentation-ready final-year project for the **Master of Computer Applications (MCA)**.

Integrating traditional Computer Vision (OpenCV) filters with deep learning classification networks (Keras CNNs & PyTorch EasyOCR), this web platform converts handwritten letters, words, sentences, and complete scanned notebook pages into clean editable digital formats. Database operations are audited in SQLite, and output files compile client-side as TXT, PDF, and DOC format sheets.

Project Objectives

  • Develop CNN classification configurations for EMNIST characters.
  • Configure character contour-bounding threshold segmentation algorithms via OpenCV.
  • Incorporate page-level handwriting translation pipelines using EasyOCR.
  • Save auditable query logs using SQLite database connections.
  • Export scanned results to TXT, PDF, and DOCX document formats.
